League Analysts in Training

I built League Analysts: Meta Miner, an end-to-end analytics pipeline that ingests Riot Match-V5 data and transforms it into per-patch win/pick/ban rates for every champion. The system handles API rate limits with retry/backoff, normalizes patch versions, and exports reproducible CSVs and visuals. On top of this dataset, I engineered features (optional logit transforms, standardization, √games weighting) and trained K-Means models to generate patch-specific tierlists (S/A/B/…), selecting K via elbow & silhouette diagnostics and checking stability across seeds. I then analyzed meta shifts across patches (bump charts, heatmaps, “movers”) and applied the tiers to real drafts (e.g., T1 vs. KT) to show when raw champion strength aligns with outcomes and when scaling/synergy or player mastery overrides the baseline. Result: a scalable, explainable workflow that turns raw API data into actionable, competitive insights.eper insights into competitive dynamics and tournament structure.
